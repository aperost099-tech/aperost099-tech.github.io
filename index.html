Why does my browser show a security warning?

This occurs because the download count is still building. If your browser says the file is unsafe, click Keep file to continue. The file is completely verified and safe.

What about the Windows SmartScreen warning?

Windows may show a blue warning screen. Click More info, then Run anyway. This is normal for newly released indie games establishing a reputation with Microsoft.

The installation does not complete?

Regional API restrictions may interfere with our servers. Using any VPN during the installation process will resolve this immediately.

Is the game safe to download?

Yes. We are a small two-person studio, and our files are manually reviewed for safety. Security systems simply need time to recognize new indie developers.
